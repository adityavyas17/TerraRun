# TerraRun

## Features

- User signup and login
- Real-time run tracking using device GPS
- Run timer, distance, and average speed
- Save completed runs to backend
- Run history page
- Leaderboard page
- Home and profile stats
- Map-based tracking using Flutter Map

## Tech Stack

### Frontend
- Flutter
- Dart
- Flutter Map
- Geolocator

### Backend
- FastAPI
- Python
- Uvicorn
- PostgreSQL
